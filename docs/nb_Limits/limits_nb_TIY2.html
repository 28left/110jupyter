Estimating limits¶

We first generate a random function \(f\) and pick a number \(r\) between \(-2\) and \(2\).

f = function('random')
r = 1.2

display(Markdown("Function $f(x)$ set, $r = $"+ str(r)))

Function $f(x)$ set, $r = $1.2

Next, we generate a table of values of \(f\) around \(r\):

x f(x)
0 1.100 -5.19062035960865
1 1.190 -5.34790661424688
2 1.199 -5.36297575209075
3 1.201 -5.36630908619569
4 1.210 -5.38124071921730
5 1.300 -5.52472852893498

What do the values \(f(x)\) suggest would be a reasonable estimate for \(\lim_{x\to r} f(x)\)?

Let’s confirm our estimate by formally computing the limit.

sp.limit(f(x),x,1.2)
\[\displaystyle -5.36464311358791\]
