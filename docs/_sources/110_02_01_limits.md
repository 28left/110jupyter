## Limit of a Function

+++

### Definition

The function $f(x)$ has a limit of L as $x$ approaches $a$,

$$ \lim_{x\to a} f(x) = L$$

if the value of $f(x)$ can be made as close to the number $L$ as we please by taking x values *sufficiently close* to, but not equal to, $a$.
